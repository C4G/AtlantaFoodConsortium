## Available chart components

- [AreaChartComponent](/documentation/features/charts/area-chart) - Smooth filled area chart for continuous data like monthly trends.
- [BarChartComponent](/documentation/features/charts/bar-chart) - Horizontal or vertical bar chart with pagination for long datasets.
- [LineChartComponent](/documentation/features/charts/line-chart) - Single or multi-line chart for comparing metrics over time.
- [DonutChart](/documentation/features/charts/donut-chart) - Proportional breakdown chart with a hollow centre and legend.
- [KPICard](/documentation/features/charts/kpi-card) - Metric summary card with icon, value, label, and trend arrow.
- [ChartInfoTooltip](/documentation/features/charts/chart-info-tooltip) - Info icon that shows a tooltip with chart description and key metric definitions.

> **Info:** All charts use Recharts
> Every chart component is built on top of Recharts and is fully responsive via `ResponsiveContainer`. Pass data as a plain array of objects.
